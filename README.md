FLUX - архитектура
---------------------
Задание по архитектуре для ШРИ-2018

* [DEMO]()

**Production**
```
npm i
npm run build
```
**Development**
```
npm i
npm run start
```
Открываем по адресу http://localhost:9000

**Технологии**
* сборка Webpack
* препроцессор SASS
* eslint, csscomb (linter)
* Babel
